üí° svg properties used to create bar charts
  • The svg element is an HTML element, the elements inside the svg element are SVG elements
  • Default units for SVG coordinates are in pixels so it is customary omit the units
  • SVG coordinate frames are relative to the svg element upper left corner
  • To ensure shapes are visible, set all needed attributes
  • When using style for setting attributes be sure that it is supported by the browser
  • Use the rect SVG element for creating bar charts
  • Use the rect x, y, width, height to place and size the bars

1. Bars with SVG rect and inline style attribute

Use the example code below and create a bar chart in the answer for the following values: [200, 150, 100, 50]
<!-- bar of 300px with text -->
<svg width="300px" height="300px">
  <rect x="10" y="10" width="100" height="30" fill="orangered"></rect>
  <text x="20" y="30" style="font-size: 12px; fill: black;">100px bar</text>
</svg>

2. Bars with SVG rect and inline style attribute with CSS inheritance

Rewrite the code of ex2.1 adding an outer group with the g SVG element and use CSS inheritance to minimize CSS redundancy on the rect elements used to plot the bars.
<svg width="300px" height="300px">
  <g style="fill: orangered;">
    <rect x="10" y="10" width="100" height="30"></rect>
    <text x="20" y="30" style="font-size: 12px; fill: black;">100px bar</text>
  </g>
</svg>
