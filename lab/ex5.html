💡 Creating bar charts with divs using d3 select and append mechanism
  • With div elements you want to point d3 to a parent div to hold the chart
  • d3.select() allows to select elements in the page by id with a CSS selector
  • You can use the chaining syntax to configure and style attributes
  • selection.text() sets the innerHTML text for the selected HTML element
  • selection.attr() sets one class attribute of the selected HTML element
  • selection.style() sets one style attribute of the selected HTML element
  • Using the d3 data join requires the most set-up but scales with the data
Use the example code below and create a bar chart in the answer for the following values: [200, 150, 100, 50]. You will need to install d3 with npm install d3.
<style>
div.bar {
  background-color: orangered;
  margin-bottom: 6px;
  text-align: right;
  padding-right: 5px;
}
</style>

<!-- install and load d3 library -->
<script src="./node_modules/d3/dist/d3.min.js"></script>

<div id="chart"></div>

<script>
var data = [100];

d3.select('#chart')
  .selectAll('div')
  .data(data)
  .enter()
  .append('div')
  .attr('class', 'bar')
  .style('width', d => d + 'px')
  .text((d, i) => d + 'px');
</script>
