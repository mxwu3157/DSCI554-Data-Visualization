üí° div properties used to create bar charts
  • When graphing in the browser you specify the units in pixels with px, however units can be specified in other units, e.g., in or mm.
  • HTML elements can be specified as absolute or relative. The default is relative to the parent.
  • When using div elements to make horizontal bar charts we can use the margin-bottom to separate the bars.
    • You can use the inspect an element in DevTools to see how margin, border and padding are set.
  • By default a div will fill the space available horizontally
  • Setting the background-color on a div will color the whole space the div occupies
  • div does not inherit the border properties.
