1. Implement a dashboard with Bootstrap

Follow these steps to implement this Dashboard:
  1. Download the Bootstrap Examples here
  2. Unzip the archive, open the index file under dashboard
  3. Check that the dashboard works by opening dashboard/index.html
  4. Copy the dashboard folder to dashboard
  5. Initialize npm and installs bootstrap in the dashboard folder: 
    cd dashboard
npm init
npm install --save bootstrap
  6. Adjust the paths for bootstrap.min.css and bootstrap.bundle.min.js in dashboard/index.html to use the files from node_modules: 
    ./node_modules/bootstrap/dist/css/bootstrap.min.css
./node_modules/bootstrap/dist/js/bootstrap.bundle.min.js
  7. Check that the dashboard works by opening dashboard with LiveServer
