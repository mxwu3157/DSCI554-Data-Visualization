Introduction

Using selection.data() in the data join with groups

When data is loaded from an API, visualizations need to be generated dynamically. We are going to see in this page how the d3 data join can be used to generate tables. To use the d3 data join mechanism in a scalable way one needs to be able to put the data in a matrix form and `group` the data by row. In addition, special care needs to be taken for the table headers.
üí° Handling groups with d3.selectAll followed by selection.selectAll
To realize tables you can use selection.data() with groups of data where each group corresponds to one row
This is explained in the selection.data() documentation:

# selection. data([ data[, key]]) <>

Joins the specified array of data with the selected elements, returning a new selection that represents the update selection: the elements successfully bound to data. Also defines the enter and exit selections on the returned selection, which can be used to add or remove elements to correspond to the new data. The specified data is an array of arbitrary values (e.g., numbers or objects), or a function that returns an array of values for each group. When data is assigned to an element, it is stored in the property __data__, thus making the data ‚Äústicky‚Äù and available on re-selection.

The data is specified for each group in the selection. If the selection has multiple groups (such as d3.selectAll followed by selection.selectAll ), then data should typically be specified as a function. This function will be evaluated for each group in order, being passed the group‚Äôs parent datum ( d, which may be undefined), the group index ( i), and the selection‚Äôs parent nodes ( nodes), with this as the group‚Äôs parent element.

  border-collapse: collapse;

  border: 1px solid black;

<script src="./node_modules/d3/dist/d3.min.js"></script>

<script>
var matrix = [
  ['th00', 'th01'],
  ['td10', 'td11'],
  ['td20', 'td21']];

d3.select('#ex_1_4')
  .selectAll('tr')
  .data(matrix)
  .enter()
  .append('tr')
  .style('font-weight', (d, i) => {
    return (i == 0) ? 'bold' : 'normal';
  })
  .selectAll('td')
  .data(d => d)
  .enter()
  .append('td')
  .text(d => d);
</script>
