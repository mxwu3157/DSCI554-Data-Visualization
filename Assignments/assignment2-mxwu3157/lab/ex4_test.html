Vega-Lite demo

Vega-Lite is a high-level grammar of interactive graphics. It provides a concise, declarative JSON syntax to create an expressive range of visualizations for data analysis and presentation.
We demonstrate Vega-Lite format, the example gallery, and discuss the following from the Vega-Lite Ecosystem:
  • Altair exposes a Python API for building statistical visualizations that follows Vega-Lite syntax.
  • Altair wrapper in R
  • Vega-Lite API is a JavaScript API for creating Vega-Lite JSON specifications.
  • Vega-Embed, a convenience wrapper for Vega and Vega-Lite.
We demonstrate the Simple Bar Chart example:
  • Discuss the spec
  • View the chart and edit the spec in the online editor and learn to export as png, svg, and add the chart in the page as images and as svg code
  • Demonstrate how to use Vega-Embed in the page
