# ex10: Observable notebook

## 😜 Exercise 0

1. load `data/iris.csv` in the notebook
2. Create a summary table using the
[summary table notebook](https://observablehq.com/@observablehq/summary-table)
3. Transform the data using the code below and create a summary table of `irisTyped`

```js
irisTyped = iris.map(d => {
  return {
    sepal_length: +d.sepal_length,
    sepal_width: +d.sepal_width,
    petal_length: +d.petal_length,
    petal_width: +d.petal_width,
    species: d.species
  }
});
```
